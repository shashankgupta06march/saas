Integration Code

To add this chatbot to your own website, copy this code:

<script>
  (function() {
    var script = document.createElement('script');
    script.src = 'http://localhost:8081/widget.js';
    script.setAttribute('data-chatbot-id', '1');
    document.body.appendChild(script);
  })();
</script>

Note: Replace '1' with your actual chatbot ID and update the URL for production.
